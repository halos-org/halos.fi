HaLOS

A container-first operating system for Raspberry Pi

Get Started Documentation

Web Management

Manage your system through an intuitive web interface. Cockpit-based administration with single sign-on via Authelia.

Container Apps

Install and manage applications from a curated app store. Each app runs in its own container for isolation and easy updates.

Easy Setup

Flash an SD card, boot, and connect. HaLOS is ready to use in minutes with sensible defaults and guided first-boot configuration.

Extensible

Built on Debian with standard package management. Add your own containers, customize the dashboard, and integrate with existing tools.
